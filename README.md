theo
====

Theo is a C++ mini file archiver

Usage:

```
theo [option] [argument]
-h                Print usage help
-c [output]       Crushing mode (Archiving)
-x [target]       Extraction mode
-l [target]       List files in archive
-i [target]       Extract specific file from archive

```
### Dependencies

boost::filesystem
